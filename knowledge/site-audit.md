## 1. Техническая база

| Параметр | Значение |
|---|---|
| **CMS** | WordPress (`/wp-json/`) |
| **Конструктор** | Elementor + Elementor Pro |
| **SEO** | Yoast SEO (JSON-LD `@graph`, sitemap) |
| **Сервер** | nginx/1.28.0, HTTP/2, HSTS 1 год |
| **Язык** | `ru-RU`, viewport есть |
| **Canonical** | На главной — есть |
| **Schema.org** | WebPage, WebSite, Organization, BreadcrumbList, ImageObject |
| **Прочее** | Cookie Notice, PHPSESSID, ~50 script / ~29 CSS на главной |

### Что работает

- HTTPS и HSTS
- Yoast отдаёт Organization schema
- `sitemap_index.xml` живой (7 подкарт)
- `robots.txt` закрывает служебные пути
- Мобильный viewport

### Критичные проблемы

1. **Три H1 на главной**, один — placeholder `Главный заголовок h1`
2. **Кэш отключён:** `cache-control: no-store, no-cache` + PHPSESSID
3. **robots.txt от Bitrix** на WordPress-сайте (`/bitrix/`, `/upload/`)
4. **Нет Product / LocalBusiness schema** на коммерческих страницах
5. **Пустой meta description** на `/krashenyj-planken/` (204 показа GSC / 90 дн.)
6. **HTML-сущности в meta** (`&quot;` на `/o-kompanii/`)
7. **16 постов блога** с дублирующими slug про «восстановление полов» при разном контенте
8. **Sitemap:** опечатка `project-categoties-sitemap.xml`, индексируется `category/uncategorized/`
9. **Двойные H1** на большинстве лендингов (шаблон Elementor)
